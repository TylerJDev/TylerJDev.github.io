Twitter GUI

Imagine using twitter on a customizable interface, that is the goal of Twitter Gui - To be able to easily access twitter with a range of different features.

If Twitter was a Internet Relay Chat (IRC) -
Twitter can be a place where time can disappear. I wanted to make an application that would allow me to easily use twitter via a command line - That's how Twitter Gui was born. Using Tkinter and Python I developed an interface to hold twitter data, this allowed me to add all sorts of cool features that twitter doesn't come with natively.
